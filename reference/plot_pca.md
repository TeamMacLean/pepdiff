# plots a pca on the treatment, seconds, bio-rep

Performs and draws a PCA plot with four panels, PCA with sample names
coloured by treatment, seconds and biorep and a scree plot of the PCA
dimensions

## Usage

``` r
plot_pca(df)
```

## Arguments

- df:

  dataframe, typically from \`import_data()\`

## Value

ggplot2 plot
