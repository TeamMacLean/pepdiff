# plot histogram of fold change distribution for a comparison

plot histogram of fold change distribution for a comparison

## Usage

``` r
plot_fc(df, log = FALSE, base = 2)
```

## Arguments

- df:

  result dataframe, typically from \`compare()\`

- log:

  log the fold change values

- base:

  base of the log, if used

## Value

ggplot2 plot
