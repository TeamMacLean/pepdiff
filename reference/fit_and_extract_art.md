# Fit ART and extract contrasts for one peptide

Fit ART and extract contrasts for one peptide

## Usage

``` r
fit_and_extract_art(data, response, factors, compare, peptide_id)
```

## Arguments

- data:

  Data for one peptide

- response:

  Name of the response variable

- factors:

  Character vector of factor names

- compare:

  Which factor to compare

- peptide_id:

  Peptide identifier

## Value

A list with model fit results and contrasts
