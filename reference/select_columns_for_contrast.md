# Select columns for contrast (legacy)

Select columns for contrast (legacy)

## Usage

``` r
select_columns_for_contrast(
  l,
  treatment = NA,
  t_seconds = NA,
  control = NA,
  c_seconds = NA
)
```

## Arguments

- l:

  List of data and row_info

- treatment:

  Treatment name

- t_seconds:

  Treatment seconds

- control:

  Control name

- c_seconds:

  Control seconds

## Value

List with treatment and control matrices
